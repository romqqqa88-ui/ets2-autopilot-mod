## Введение

Autopilot Mod - это модификация для Euro Truck Simulator 2, которая добавляет функцию автопилота. Мод позволяет вашему грузовику автоматически следовать по маршруту, заданному в навигаторе игры, без вашего непосредственного участия в управлении.

**Основные возможности:**
- Автоматическое движение по установленному маршруту
- Адаптация скорости к дорожным условиям
- Обнаружение и объезд препятствий
- Интеграция со встроенным навигатором игры
- Настраиваемые параметры управления

## Установка

### Способ 1: Через папку модов (рекомендуется)

1. Скачайте архив с модом
2. Распакуйте архив в папку модов ETS 2:
   ```
   C:\Users\[ВашеИмя]\Documents\Euro Truck Simulator 2\mod\
   ```
3. Убедитесь, что в папке появился файл `autopilot_mod.scs` или папка `autopilot_mod`

### Способ 2: Через Steam Workshop (если доступно)

1. Подпишитесь на мод в Steam Workshop
2. Запустите игру
3. Мод автоматически загрузится и будет доступен в меню управления модами

## Активация

1. Запустите Euro Truck Simulator 2
2. Перейдите в меню **Управление модами** (Mod Manager)
3. Найдите в списке **"Autopilot Mod"**
4. Перетащите мод из левой колонки (Available mods) в правую (Active mods)
5. Нажмите **"Применить изменения"** (Apply changes)
6. Начните новую поездку или загрузите существующее сохранение

## Использование

### Быстрый старт

1. **Установите маршрут** в навигаторе игры
2. **Включите автопилот** нажатием **CTRL + A**
3. **Наблюдайте** как грузовик самостоятельно следует по маршруту
4. **Будьте готовы** взять управление в любой момент

### Горячие клавиши

| Клавиша | Действие | Описание |
|---------|----------|----------|
| **CTRL + A** | Включить/выключить автопилот | Основная функция управления |
| **CTRL + S** | Показать/скрыть панель управления | Открывает настройки автопилота |
| **CTRL + D** | Экстренная остановка | Мгновенное торможение и отключение |
| **CTRL + +** (NumPad) | Увеличить скорость | +5 км/ч к целевой скорости |
| **CTRL + -** (NumPad) | Уменьшить скорость | -5 км/ч от целевой скорости |
| **CTRL + C** | Переключить режим камеры | Для наблюдения за работой автопилота |

### Панель управления

Панель управления (F6) содержит следующие элементы:

1. **Кнопка включения** - активация/деактивация автопилота
2. **Ползунок скорости** - установка максимальной скорости (30-130 км/ч)
3. **Ползунок дистанции** - расстояние до впереди идущего транспорта (20-100 м)
4. **Ползунок агрессивности** - стиль управления (0.1-1.0)
5. **Кнопка закрытия** - скрыть панель

## Настройки

### Основные параметры (через конфигурационный файл)

Для продвинутых пользователей доступна ручная настройка в файле:
```
autopilot_mod/def/autopilot_config.sui
```

**Важные параметры:**

```sii
default_speed_limit: 90.0        -- Ограничение скорости по умолчанию
default_follow_distance: 50.0    -- Дистанция следования
aggressiveness: 0.5              -- Агрессивность управления
enable_lane_change: true         -- Разрешить смену полосы
stop_at_traffic_lights: true     -- Останавливаться на светофорах
```

### Консольные команды

Откройте консоль игры (клавиша **~**) и введите:

- `autopilot_toggle` - переключить автопилот
- `autopilot_speed 80` - установить скорость 80 км/ч
- `autopilot_distance 40` - установить дистанцию 40 метров
- `autopilot_aggressiveness 0.7` - установить агрессивность 0.7
- `autopilot_debug on` - включить отладочный режим

## Решение проблем

### Проблема: Автопилот не включается
**Решение:**
1. Проверьте, активирован ли мод в Mod Manager
2. Убедитесь, что установлен пункт назначения в навигаторе
3. Проверьте, что грузовик находится на дороге (не в канаве)
4. Перезагрузите сохранение

### Проблема: Грузовик съезжает с дороги
**Решение:**
1. Уменьшите агрессивность управления (0.3-0.5)
2. Убедитесь, что маршрут в навигаторе корректен
3. Проверьте, не установлены ли другие моды, влияющие на физику

### Проблема: Низкая производительность
**Решение:**
1. Отключите отладочную информацию в настройках
2. Уменьшите дальность обнаружения препятствий
3. Закройте панель управления (F6)

### Проблема: Автопилот не останавливается на светофорах
**Решение:**
1. Убедитесь, что опция `stop_at_traffic_lights` включена
2. Проверьте, что мод совместим с вашей версией игры
3. Обновите мод до последней версии

## Часто задаваемые вопросы

### В: Совместим ли мод с другими модами?
**О:** Да, мод совместим с большинством модов, но могут быть конфликты с модами, изменяющими:
- Физику автомобиля
- Навигационную систему
- Управление

### В: Работает ли автопилот с прицепами?
**О:** Да, автопилот учитывает наличие прицепа и корректирует управление.

### В: Можно ли использовать автопилот в онлайн-режиме (TruckersMP)?
**О:** Нет, TruckersMP запрещает использование модов, изменяющих геймплей.

### В: Насколько безопасен автопилот?
**О:** Автопилот является вспомогательной системой. Всегда следите за дорогой и будьте готовы взять управление.

### В: Потребляет ли автопилот больше ресурсов компьютера?
**О:** Минимально. При возникновении проблем с производительностью отключите отладочный режим.

### В: Как отключить голосовые уведомления?
**О:** В конфигурационном файле установите `enable_voice_notifications: false`

### В: Можно ли настроить автопилот под свой стиль вождения?
**О:** Да, используйте параметры агрессивности, дистанции и скорости для тонкой настройки.

## Безопасность и рекомендации

1. **Не оставляйте компьютер без присмотра** при включенном автопилоте
2. **Всегда следите за дорогой** и будьте готовы взять управление
3. **Отключайте автопилот** в сложных условиях:
   - Плотный городской трафик
   - Сложные развязки
   - Плохие погодные условия (сильный дождь, снег)
   - Строительные участки
4. **Регулярно сохраняйтесь** перед использованием новых функций
5. **Сообщайте об ошибках** для улучшения мода
