# Описание файла hud_elements.dds
# Это текстовое описание вместо реального DDS файла
# В реальном моде здесь должен быть DDS файл с графическими элементами HUD

Формат: DDS (DirectDraw Surface)
Размер: 512x512 пикселей
Цветовая модель: RGBA8
MIP-уровни: 10

Содержимое файла включает следующие элементы:

1. Иконка автопилота (64x64)
   - Состояния: выключен, включен, активен, ошибка
   - Позиция в текстуре: (0,0)-(64,64)

2. Индикатор скорости (128x64)
   - Шкала скорости от 0 до 150 км/ч
   - Позиция: (64,0)-(192,64)

3. Индикатор маршрута (256x128)
   - Отображение текущего маршрута
   - Позиция: (0,64)-(256,192)

4. Индикатор препятствий (128x128)
   - Отображение ближайших транспортных средств
   - Позиция: (256,0)-(384,128)

5. Индикатор состояния систем (256x64)
   - Статус: навигация, управление, безопасность
   - Позиция: (0,192)-(256,256)

6. Элементы интерфейса настроек
   - Кнопки, ползунки, переключатели
   - Позиция: (256,128)-(512,256)

7. Текстовые символы (256x256)
   - Цифры 0-9, буквы A-Z, специальные символы
   - Позиция: (0,256)-(256,512)

Примечание: Для создания реального DDS файла используйте инструменты:
- NVIDIA Texture Tools
- Paint.NET с плагином DDS
- Photoshop с плагином NVIDIA DDS
