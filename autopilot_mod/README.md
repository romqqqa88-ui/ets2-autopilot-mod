# Autopilot Mod для Euro Truck Simulator 2

## Описание

Мод добавляет функцию автопилота в Euro Truck Simulator 2, позволяя грузовику автоматически следовать по маршруту, заданному во встроенном навигаторе игры. Автопилот использует данные навигатора для определения пути, контролирует скорость, рулевое управление и реагирует на препятствия.

## Основные возможности

- **Автоматическое движение по маршруту** - грузовик самостоятельно следует к точке назначения
- **Интеграция с навигатором** - использует встроенный навигатор игры для получения маршрута
- **Адаптивное управление скоростью** - автоматически регулирует скорость в зависимости от дорожных условий
- **Обнаружение препятствий** - обнаруживает другие транспортные средства и объекты на дороге
- **Безопасное торможение** - автоматическое торможение при опасности столкновения
- **Настраиваемые параметры** - гибкая настройка под стиль вождения пользователя
- **Пользовательский интерфейс** - удобная панель управления с настройками

## Требования

- Euro Truck Simulator 2 версии 1.52 или выше
- Поддержка модов в игре (активирована в настройках)

## Установка

1. Скачайте архив с модом
2. Распакуйте архив в папку модов ETS 2:
   - Обычный путь: `Documents/Euro Truck Simulator 2/mod/`
3. Запустите игру
4. В меню управления модами активируйте "Autopilot Mod"
5. Начните новую поездку или загрузите существующую

## Использование

### Основные команды

- **CTRL + A** - Включить/выключить автопилот
- **CTRL + S** - Показать/скрыть панель управления
- **CTRL + D** - Экстренная остановка
- **CTRL + +** (NumPad) - Увеличить целевую скорость
- **CTRL + -** (NumPad) - Уменьшить целевую скорость

### Настройка через интерфейс

1. Нажмите CTRL+S для открытия панели управления
2. Настройте параметры:
   - Ограничение скорости
   - Дистанцию следования
   - Агрессивность управления
3. Нажмите "Включить автопилот" для активации

### Консольные команды

- `autopilot_toggle` - включить/выключить автопилот
- `autopilot_speed <значение>` - установить скорость (30-130 км/ч)
- `autopilot_ui` - показать/скрыть интерфейс

## Настройка параметров

Основные параметры можно настроить в файле `def/autopilot_config.sui`:

- `default_speed_limit` - ограничение скорости по умолчанию
- `default_follow_distance` - дистанция до впереди идущего транспорта
- `aggressiveness` - агрессивность управления (0.1 - осторожно, 1.0 - агрессивно)
- `enable_lane_change` - разрешить автоматическую смену полосы

## Решение проблем

### Автопилот не включается
1. Убедитесь, что мод активирован в меню управления модами
2. Проверьте, установлен ли пункт назначения в навигаторе
3. Убедитесь, что грузовик находится на дороге

### Грузовик не следует по маршруту
1. Проверьте, активирован ли маршрут в навигаторе
2. Убедитесь, что автопилот включен (индикатор на панели)
3. Проверьте настройки дистанции следования

### Проблемы с производительностью
1. Отключите отладочную информацию в настройках
2. Уменьшите дальность обнаружения препятствий
3. Закройте панель управления (F6)

## Безопасность

Автопилот является вспомогательной системой и не заменяет внимательность водителя. Рекомендуется:
- Всегда следить за дорогой
- Быть готовым взять управление на себя в любой момент
- Не использовать автопилот в сложных погодных условиях
- Отключать автопилот в городах и на сложных развязках

## Разработка

### Для разработчиков модов

Мод использует стандартный API ETS 2 для скриптов. Основные компоненты:

1. **autopilot.lua** - основной модуль, координация работы всех систем
2. **speed_controller.lua** - PID-контроллер для управления скоростью
3. **obstacle_detector.lua** - система обнаружения и классификации препятствий
4. **navigation_integration.lua** - получение данных маршрута из навигатора

### Тестирование

Для тестирования используйте скрипт `test_autopilot.lua`. Запустите тесты через консоль игры:
```
autopilot_test_run
```
